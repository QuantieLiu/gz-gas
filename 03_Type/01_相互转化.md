
#### String -> int
 
``` 
i=Integer.parseInt(s);
直接使用静态方法，不会产生多余的对象，但会抛出异常

i=Integer.valueOf(s).intValue();
//Integer.valueOf(s) 
相当于 new Integer(Integer.parseInt(s))，也会抛异常，但会多产生一个对象
``` 

#### int -> String
 
``` 
s=i+"";   
//会产生两个String对象

s=String.valueOf(i);
直接使用String类的静态方法，只产生一个对象
```
